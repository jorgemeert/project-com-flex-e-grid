# 🚲 Bikcraft - Landing Page de Exercício

Este projeto é um exercício prático de desenvolvimento web focado em **CSS Grid Layout** e **Flexbox**. O objetivo foi construir uma interface semântica e organizada para a apresentação de uma marca de bicicletas elétricas premium.

## 🚀 Tecnologias Utilizadas

* **HTML5**: Estruturação semântica de componentes.
* **CSS3**: Estilização avançada com foco em sistemas de layout modernos.
    * **Flexbox**: Utilizado para o alinhamento e distribuição do menu de navegação.
    * **CSS Grid**: Utilizado para a estrutura complexa da seção principal e da grade de vantagens.

## 🛠️ Destaques do Projeto

* **Grid Complexo**: A seção de "Vantagens" utiliza posicionamento de linhas e colunas para criar um layout assimétrico onde o título ocupa múltiplas linhas.
* **Posicionamento Estratégico**: Uso de `place-self` e `grid-template-areas` implícito para garantir que textos e botões fiquem perfeitamente alinhados à direita e à esquerda conforme o design.
* **Semântica**: Uso correto de tags como `<footer>`, `<h1>`, `<h2>` e botões para garantir acessibilidade básica.
